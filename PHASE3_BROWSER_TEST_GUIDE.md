### Step 9: 验证后端数据（开发者工具）

1. 打开浏览器开发者工具（F12）
2. 切换到 Network 标签
3. 在 Step 2 点击提交按钮
4. 查找 `POST /api/interview/session` 请求

**检查 Request Payload：**
```json
{
  "mode": "RESUME",
  "jobTitle": "Frontend Developer",
  "difficulty": "MEDIUM",
  "resumeId": "clx...",
  "customKnowledgePoints": [
    "React Hooks Advanced",
    "TypeScript Generics",
    "你添加的自定义知识点",
    ...
  ]
}
```

**检查 Response：**
```json
{
  "message": "Interview session created successfully",
  "data": {
    "sessionId": "clx...",
    "greeting": "你好！我是你的 AI 面试官..."
  }
}
```

**预期结果：**
- ✅ Payload 包含 `customKnowledgePoints` 数组
- ✅ 数组包含你编辑后的知识点
- ✅ 响应包含 `sessionId` 和 `greeting`
- ✅ HTTP 状态码 201
