## 可能的错误情况

### 错误 1：仍然超时
**症状：** 等待 120 秒后显示超时错误

**可能原因：**
1. AI API 响应过慢
2. 网络连接不稳定
3. PDF 文件过大或格式复杂

**解决方案：**
1. 检查网络连接
2. 尝试上传更小的文件（< 2MB）
3. 检查后端日志查看详细错误
4. 验证 DeepSeek API Key 是否有效

### 错误 2：格式错误
**症状：** 显示 "❌ 格式错误，仅支持 PDF 格式！"

**原因：** 上传了非 PDF 文件

**解决方案：** 确保上传的是 PDF 格式文件

### 错误 3：文件过大
**症状：** 显示 "文件大小不能超过 10MB"

**原因：** 文件超过 10MB 限制

**解决方案：** 压缩 PDF 文件或使用更小的文件

## 调试方法

### 1. 查看浏览器控制台
1. 按 F12 打开开发者工具
2. 切换到 Console 标签
3. 查看是否有错误消息

### 2. 查看网络请求
1. 按 F12 打开开发者工具
2. 切换到 Network 标签
3. 上传文件
4. 查找 `POST /api/resume/extract` 请求
5. 检查：
   - Status: 应该是 200 或 201
   - Time: 应该在 120 秒内
   - Response: 应该包含 personalInfo, highlights, knowledgePoints

### 3. 查看后端日志
在后端终端中查看日志输出，应该看到：
```
[Resume Extract] Started for user xxx
[Resume Extract] Completed in xxxms
```
